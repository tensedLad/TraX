# Contributing to TraX

Thanks for your interest! Here's how to get started:

## Setup
1. Fork this repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/trax.git`
3. Install dependencies: `npm install`
4. Create a `.env` file based on `.env.example`
5. Run locally: `npm run dev`

## Making Changes
- Create a new branch: `git checkout -b feature/your-feature-name`
- Make your changes
- Commit with clear messages: `git commit -m "feat: add portfolio chart view"`
- Push: `git push origin feature/your-feature-name`
- Open a Pull Request on the main repo

## Guidelines
- Don't break existing functionality
- Keep code clean and readable
- One feature per PR please
